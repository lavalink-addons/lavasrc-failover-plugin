# Contributing to LavaSrc Failover Plugin

Thanks for your interest in contributing! Here's how you can help.

## Getting Started

1. Fork the repository
2. Clone your fork:
   ```bash
   git clone https://github.com/YOUR_USERNAME/lavasrc-failover-plugin
   cd lavasrc-failover-plugin
   ```
3. Create a new branch:
   ```bash
   git checkout -b feat/your-feature-name
   ```

## Requirements

- JDK 17+
- Gradle 8+

## Building

```bash
gradle jar
```

The compiled `.jar` will be at `build/libs/`.

## Submitting a Pull Request

1. Make sure your code compiles without errors
2. Keep commits clean and descriptive
3. Open a Pull Request against the `main` branch
4. Describe what your PR does and why

## Reporting Issues

If you found a bug or have a feature request, open an [issue](https://github.com/doyimmiuink/lavasrc-failover-plugin/issues) with:

- A clear title
- Steps to reproduce (if a bug)
- Expected vs actual behavior
- Lavalink and LavaSrc versions you're using

## Code Style

- Kotlin only
- Follow existing code structure
- Keep things simple and readable
